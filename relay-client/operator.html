One-Click Start

  -PublicBaseUrl "https://codex.flying-agents.com" `
  -SessionSecret "PUT_A_LONG_RANDOM_SECRET_HERE"

Different-Network Setup

For phone access from another network:

1. Keep app-server local:
ws://127.0.0.1:4500

2. Expose only the relay.

3. Set relay env vars before start:
CODEX_RELAY_HOST=0.0.0.0
CODEX_RELAY_BASE_URL=https://YOUR_PUBLIC_HOST
CODEX_RELAY_SESSION_SECRET=your-random-secret

4. Use HTTPS/WSS for the phone path.

5. Do not expose raw app-server directly.
